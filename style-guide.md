# Christine's BioSite - Design System Documentation

## Overview
This bioSite is built using **Atomic Design principles** with Christine's custom style guide, featuring the Rockwell typeface and a beautiful blush/purple/navy color palette.

---

## Style Guide

### Color Palette

| Color | Hex Code | Usage |
|-------|----------|-------|
| **Navy** | `#15173D` | Navigation bar, paragraph text (header & p)|
| **Purple** | `#982598` | Navigation links, buttons, titles (h2) |
| **Blush** | `#F1E9E9` | Anchor tag, site title (a & h1) |

### Typography
**Primary Font**: Rockwell (serif font with strong character)
**h1 Font**: Georgia (transitional serif typeface)
